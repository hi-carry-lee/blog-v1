# 使用 resend 发送邮件

## resend 验证域名

1. 如果不验证自己的域名，那么只能向自己的邮箱发送邮件，from 只能是 resend 的测试邮箱
2. 验证之后，只能用以域名为后缀的邮箱地址，邮箱地址不存在也可以，比如 `noreply@xxxx.com`

## vercel KV 存储

1. 当前使用它来实现速率限制，但是 vercel 的 KV 是基于 upstash，它有 7 天不活跃删除策略；
2. 改用数据库验证作为后备策略；
3. 也可以尝试用定时任务去定时访问，避免失效；

## 代码实现注意事项

1. 速率限制，以及发送邮件都需要使用 `try-catch` 捕获异常，方便调试 bug；
